##Правила игры (кратко)
У каждого игрока есть поле 10x10 клеток(нумерация от A1 до J10), на котором размещены корабли:
1 четырехпалубный
2 трехпалубных
3 двухпалубных
4 однопалубных

Корабли не могут касаться друг друга (даже по диагонали).
Расстояние между кораблями минимум 1 клетка.
Игроки по очереди делают "выстрелы" по координатам поля соперника.
Если выстрел пришелся по кораблю — "попадание", если выстрел уничтожил весь корабль - "убил", иначе — "мимо".
Если игрок "попал" или "убил", то он продолжает свой ход.
Цель — первым потопить все корабли противника.

##Как будет работать ассистент 
Предоставляет интерфейс для вноса и контроля ходов.
Проверяет корректность размещения кораблей и ходов автоматически.
Не допустит "нечестных" действий (например, внесения хода "задним числом").
Показывает статус текущей партии и промежуточные итоги.
Сохраняет всю историю партий, позволяет просматривать и анализировать прошлые игры.
Автоматически рассчитывает индивидуальные рейтинги игроков и разнообразную статистику (процент побед, средняя длина партии и др.).
